Getting Started with Forms-X

Modern, reactive form utilities for Angular that embrace signals and immutability

What is Forms-X?

Forms-X is a collection of lightweight, signal-based utilities designed to make form handling in Angular simpler, more reactive, and more maintainable. It provides four key utilities:

  • SignalArray - Reactive array management with mutable-style API
  • FormTracker - Automatic dirty state tracking for forms
  • SparkAsyncValidator - Debounced async validation with loading states
  • DebounceFieldDirective - Two-way binding with configurable debounce
